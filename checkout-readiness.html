Operativ regel

Vis ønsket kjøpsreise i preview. Lov kun det som er testet i Shopify.

Previewen viser Vipps, kort, Posten/Bring og fast levering som ønsket retning. Produksjon krever Shopify-butikk, godkjente betalingskontoer, konfigurert frakt, testordre, returvilkår og opplæring.

Betaling

Vipps og kort må testes som faktiske checkout-ruter

Må settes opp Shopify Payments / kort

Bekreft bedrift, bankkonto, valuta, MVA/skatt, 2FA, korttyper, payout, capture, refund og chargebacks.

Må settes opp Vipps/MobilePay app

Opprett merchant, installer offisiell app, koble riktig merchant og test at Vipps vises for norsk checkout.

Ikke nok Shopify Payments MobilePay alene

MobilePay-kilden har kundelokasjon- og subscription-begrensninger. Bruk den som risikokilde, ikke som bevis på norsk Vipps.

Ikke lov ennå Express Checkout

Ikke planlegg Vipps Express i Shopify før dokumentert støtte finnes. Vipps/MobilePay oppgir at dette ikke er tilgjengelig i Shopify.

Må avklares Fast levering

Subscriptions krever app, vilkår, støttet gateway og full test av fornyelse, feilet betaling, endring og kansellering.

Ikke anta Vipps + abonnement

Ikke lov recurring Vipps i Shopify før valgt arkitektur er verifisert. Direkte Recurring API kan være en annen teknisk løype.

Preview Engangskjøp

Previewen viser engangskjøp og fast levering uten rabatt, frekvens eller juridiske subscription-løfter.

Posten/Bring

Fraktoppsettet må testes helt fra rate til tracking

Kundeavtale

Avklar Posten/Bring-avtale, API/app og hvilke leveringsprodukter som skal tilbys.

Variantdata

Legg inn korrekt vekt, dimensjoner, lagerlokasjon og eventuell fri-frakt terskel.

Live rates

Test at Shopify-plan/app støtter live carrier rates eller valgt shipping-rate løsning.

Hentested

Test hentested/pakkeboks på mobil og desktop, inkludert norske adresser og postnummer.

Etikett

Opprett testordre, skriv ut fraktetikett, bekreft servicekode og pakkedimensjon.

Sporing

Bekreft trackingnummer, ordrebekreftelse, e-post/SMS og kundeservicesvar ved forsinkelse.

Testordreplan

Minimumstest før lansering

Størrelse 1 + Vipps

Betaling, capture, ordrebekreftelse og refund.

Størrelse 2 + kort

Visa/Mastercard, 3DS der relevant, payout og refund.

Feilet betaling

Avbrutt Vipps, avvist kort, supporttekst og cart recovery.

Posten/Bring hentested

Rate, hentestedvelger, etikett, tracking og ordrestatus.

Hjemlevering

Test bare hvis dette tilbys i første release.

Rabatt/fri frakt

Hvis brukt: rabattkode, terskel, MVA og totalsum.

Kansellering

Kanseller før fulfilment og bekreft betalingsstatus.

Skadet vare

Reklamasjonsflyt, ny sending/refund og kundeservice-svar.

Fast levering

Kun hvis abonnement er i scope: fornyelse, pause, endring, kansellering og failed payment.

SSL, consent og opplæring

Teknisk drift må være del av leveransen

Domene / SSL

  • Domene koblet til Shopify.
  • HTTPS aktiv uten mixed content.
  • Canonical URLs satt.
  • Search Console og sitemap klare.

Vilkår

  • Kjøpsvilkår godkjent.
  • Retur/angrerett godkjent.
  • Fraktvilkår og skadet vare-prosess godkjent.
  • Subscription-vilkår hvis fast levering beholdes.

Consent

  • CMP/cookieverktøy valgt.
  • GA4 ecommerce events etter samtykke.
  • Ingen pixels før samtykke.
  • Personvern tekst oppdatert for checkout/support.

Opplæring

  • Produkter, pris og lager.
  • Ordre, refund og kansellering.
  • Fraktetikett og tracking.
  • FAQ/support, retur og rapporter.
