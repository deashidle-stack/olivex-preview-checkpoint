# OliveX Share Preview

This folder is made for drag-and-drop hosting.

Upload this folder to Netlify Drop, Vercel, Cloudflare Pages or another static host.

The root `index.html` opens the preview hub automatically. The actual website prototype is `website.html`.

Important files:

- `index.html`: preview hub
- `review-guide.html`: client-friendly review guide
- `claim-review.html`: browser-native claim/legal review page
- `production-intake.html`: production intake/checklist page
- `launch-checklist.html`: Shopify launch checklist page
- `website.html`: OliveX website prototype
- `deck/index.html`: client presentation
- `concept-board/index.html`: visual concept board
- `docs/`: project docs
- `assets/`: temporary OliveX logo assets and future producer video slot
- `deliverables/olivex-shopify-theme.zip`: current OliveX-aligned Shopify scaffold
- `shopify-theme/`: unpacked Shopify scaffold for inspection

If your host gives you a URL like:

`https://example.netlify.app`

Send that exact URL. No `/preview.html` suffix is needed for this folder.

To add the producer video later, place it at:

`assets/producer-video.mp4`

Optional poster image:

`assets/producer-video-poster.jpg`
