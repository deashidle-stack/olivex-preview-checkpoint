# Approach

## Positioning

The store should not feel like another broad wellness catalogue. It should feel like a small, serious, premium lab-backed product house.

Working concept: **Nordic Longevity Lab**.

Design language:

- Warm clinical, not cold medical
- Product-first, not generic lifestyle-first
- Batch documentation close to purchase
- Editorial typography with restrained UI density
- Mineral green, graphite, warm white, icy blue and amber data accents

## Why This Beats The Reference

Agelab has useful patterns: Norwegian trust, reviews, product education, analysis certificates and direct ecommerce CTAs. The improvement is to reduce clutter and make trust easier to scan.

Key upgrade:

- Make each product page feel like a mini clinical dossier
- Put lab data in a visible Quality Passport
- Use fewer badges, stronger hierarchy and more whitespace
- Explain dosage and documentation before dramatic benefit language
- Let the limited product range feel curated rather than small
