# OliveX Claims Review Register - 2026-06-07

Dette er et arbeidsregister for juridisk/brand-godkjenning før OliveX går live. Det endrer ikke dagens preview-copy; previewen er fortsatt en kildetekstbasert reviewflate. Registeret viser hvilke påstander som må avklares før de kan publiseres i en produksjonsbutikk.

Status:

- `OK for preview`: kan stå i review/prototype.
- `Må dokumenteres`: kan vurderes etter batchdata/COA og korrekt kontekst.
- `Må godkjennes juridisk`: må vurderes mot Mattilsynet/EU før live.
- `Ikke bruk live`: bør fjernes/erstattes i produksjonscopy.

## Operativ Regel

Live-butikken skal ikke bruke sykdoms-, behandlings-, biomarkør-, legemiddelsammenlignings- eller "uten bivirkninger"-påstander før juridisk/regulatorisk godkjenning er gjort.

Det tryggeste produksjonsprinsippet er:

1. Vis analysebevis.
2. Forklar smak, bruk, oppbevaring og opprinnelse.
3. Bruk bare godkjente helsepåstander når nøyaktige vilkår er dokumentert.
4. Håndter medisinske spørsmål med lege/farmasøyt-handoff.

## Production-Safe Copy Direction

Bruk disse retningene som utgangspunkt for produksjonscopy etter godkjenning:

| Tema | Tryggere retning |
| --- | --- |
| Fenoler | "Batchdokumentert fenolprofil" |
| Smak | "Pepperaktig og bitter sensorikk, typisk for oljer med tydelig fenolprofil" |
| Analyse | "Analysebevis for hver batch" |
| Opprinnelse | "Sporbar opprinnelse fra produsent og region" |
| Bruk | "Utviklet for daglig bruk i en enkel rutine" |
| EU claim | "Olivenoljepolyfenoler bidrar til å beskytte blodlipider mot oksidativt stress" bare hvis batchen oppfyller vilkårene og 20 g daglig kontekst vises |
| Medisinske spørsmål | "For spørsmål om sykdom, medisiner, graviditet eller behandling bør du kontakte lege eller farmasøyt." |

## Required Evidence Before Live Claims

Før noen substansielle produktpåstander går live:

- COA/analysebevis per batch.
- Bekreftet måleenhet og metode for polyfenoler/fenolprofil.
- Dokumentasjon på hydroxytyrosol og derivater hvis EU-påstanden skal brukes.
- Produktspesifikasjon for hver variant.
- Juridisk vurdering av all hero-, produkt-, FAQ-, support- og annonsetekst.
- Godkjent supportagent-kunnskapsbase.

## Support Agent Implications

Produksjonsagenten skal svare på:

- bruk
- smak/sensorikk
- oppbevaring
- betaling/frakt/retur
- hvor analysebevis finnes
- hva batchdata betyr

Produksjonsagenten skal ikke svare substansielt på:

- sykdom
- blodtrykk, diabetes, Alzheimer, fettlever, betennelse eller biomarkører
- medisininteraksjoner
- dosering for medisinsk effekt
- om kunden kan slutte eller endre medisin
